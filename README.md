\# Job Tracker System 秋招智能求职管理系统

> 一款基于Python开发的轻量化本地求职投递管理工具，支持 \*\*CLI命令行终端\*\* + \*\*Flask Web网页端\*\*，集成离线OCR岗位识别、简历-JD智能匹配打分、投递数据统计分析。



\## ✨ 功能特性

1\. \*\*投递记录管理\*\*

投递信息增删改查，归档公司名称、岗位、招聘链接、JD原文、投递状态、岗位匹配得分；CSV本地持久化存储。



2\. \*\*Tesseract-OCR 截图录入\*\*

上传招聘岗位截图，离线识别文字，自动提取岗位信息，减少手动复制JD的重复工作。



3\. \*\*简历 \& JD 智能自动打分【核心亮点】\*\*

摒弃人工填写分数，支持多种简历来源：

\- 直接粘贴简历文本

\- `.txt` / `.docx` Word简历 / `.pdf` PDF简历自动解析

内置技能关键词库，自动对比简历与岗位需求，输出0\~100匹配分数、已掌握技能、缺失技能清单，分数可保存至投递记录。



4\. \*\*数据统计与导出\*\*

自动统计各状态投递数量、面试转化率，支持导出CSV报表备份。



5\. \*\*Flask Web可视化平台\*\*

复用终端业务逻辑代码，浏览器在线查看、新增、编辑投递记录。



\## 🛠 技术栈

\- 语言：Python3.9+

\- 存储：CSV 文件（无需数据库，开箱即用）

\- Web框架：Flask

\- OCR：Tesseract-OCR + Pillow

\- 文档解析：python-docx、PyPDF2
